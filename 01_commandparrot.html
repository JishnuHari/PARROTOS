Commands

Parrot OS includes unique tools and commands that cater specifically to security, privacy, and development needs. Here are some distinctive commands and tools you can use in Parrot OS:

1. AnonSurf

- anonsurf start: Activates AnonSurf, routing all traffic through the Tor network for anonymized internet browsing.

- anonsurf stop: Stops AnonSurf and restores normal network settings.

- anonsurf status: Checks the current status of AnonSurf.

2. Parrot Tools

Parrot OS comes with a suite of custom tools grouped under the parrot-tools meta-package. You can manage these tools with specific commands:

- sudo apt-get install parrot-tools-full: Installs the full suite of Parrot OS security tools.

- sudo apt-get install parrot-tools-cloud: Installs tools specifically for cloud penetration testing.

- sudo apt-get install parrot-tools-forensics: Installs tools for digital forensics analysis.

3. Parrot Update

- sudo parrot-upgrade: A custom command that updates the system and all installed packages, combining apt-get update and apt-get dist-upgrade for convenience.

4. Firejail

Firejail is a security sandboxing tool included in Parrot OS:

- firejail firefox: Runs Firefox within a sandbox, isolating it from the rest of the system for enhanced security.

- firejail --list: Lists all running sandboxed applications.

- firejail --tree: Displays a tree of sandboxed processes.

5. Tor Tools

- torbrowser-launcher: Launches the Tor Browser, providing secure and anonymous web browsing.

- tor-control-panel: Opens the Tor control panel for managing Tor connections and settings.

6. Forensic Mode

To enable forensic mode (assuming a live boot), you need to boot with the specific option:

- Boot with forensic kernel parameter: This ensures the system does not mount any disks automatically, preserving the state of the evidence.

7. Custom Scripts and Utilities

Parrot OS includes several custom scripts for security and privacy operations:

- sudo parrot-secure: A script to enhance system security settings.

- sudo parrot-privacy: A script to adjust privacy settings across the OS.

8. WiFi Penetration Tools

- airmon-ng: Enables monitor mode on wireless interfaces.

- airodump-ng: Captures raw 802.11 frames, useful for analyzing WiFi networks.

- aircrack-ng: Cracks WEP and WPA-PSK keys.

9. Development Tools

- geany: Launches the Geany IDE, a lightweight and fast code editor.

10. System Monitoring and Management

- htop: An interactive process viewer to monitor system resources and manage processes.

- ufw: Commands to manage the Uncomplicated Firewall, enhancing system security.

These commands and tools highlight the specialized capabilities of Parrot OS, making it a powerful platform for security professionals, developers, and privacy-conscious users.
